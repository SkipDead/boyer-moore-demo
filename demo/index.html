Boyer-Moore String Search Algorithm

The Boyer-Moore string search algorithm is one of the fastest algorithms for finding a pattern in a string. It achieves its speed by comparing the text against the pattern starting at the back of the pattern, which allows the algorithm to skip many comparisons when it finds characters that do not match. The exact number of characters to skip depends on two lookup tables, which are obtained by preprocessing the pattern.
