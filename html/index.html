Boyer-Moore String Search Algorithm

The Boyer-Moore string search algorithm is one of the fastest algorithms for finding a pattern in a text. It works in two phases: First, it preprocesses the pattern into two lookup tables. Second, it compares the text to the pattern, starting at the back of the pattern, using the lookup tables to shift the comparison index whenever the current character in the pattern and text do not match. Because it compares the two strings starting from the back of the pattern, the algorithm can skip large portions of the text that could not possibly match the pattern.
